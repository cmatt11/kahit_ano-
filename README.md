# kahit ano~

Carla's personal corner of the internet. A static website that's part blog, part music diary, part photo gallery, and all vibes.

## Pages

- **Home** (`index.html`) - Landing page with previews of everything
- **Mga Tots Ko** (`tots.html`) - Blog-style thoughts with Tagalog flair
- **Music** (`music.html`) - Embedded Spotify playlists and YouTube music
- **Photos** (`photos.html`) - Pinterest-style gallery (placeholders for now)
- **Videos** (`videos.html`) - Favorite clips and comfort videos
- **About** (`about.html`) - The deeper intro
- **Links** (`links.html`) - Portfolio and social links

## Tech

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step. Just open `index.html` in a browser.

## Design

- Font: Apple system font stack (SF Pro Display, -apple-system, BlinkMacSystemFont)
- Colors: Brown, black, white base with soft pink accents
- Vibe: Minimal, cozy, personal
